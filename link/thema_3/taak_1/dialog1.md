**Alles goed?**

_Lex ontmoet een vriend op straat, Jafar._

| Lex   | Hé hallo, Jafar.                                                  |
| ----- | :---------------------------------------------------------------- |
| Jafar | Lex!**Alles** goed?                                               |
| Lex   | Ja, prima, joh. En jij?                                           |
| Jafar | Nou, niet zo goed. Ik ben ziek.                                   |
| Lex   | Ik zie het.**Wat**vervelend!**Maar**waarom loop je dan op straat? |
| Jafar | Ik heb geen paracetamol meer. Ik ga even naar de Etos.            |
| Lex   | Oké.**Beterschap**!                                               |
| Jafar | Dank je. Tot ziens!                                               |
