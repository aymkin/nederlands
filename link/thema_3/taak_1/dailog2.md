**Hoe gaat het?**

*Halina ontmoet een vriendin op straat, Mercedes.*

| Mercedes | Hé, Halina!                                                                                                                                                    |
| -------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Halina   | Ha, Mercedes. Hoe**gaat het**?                                                                                                                            |
| Mercedes | Goed hoor. Een beetje**druk**, maar dat is prima. En jij?                                                                                                 |
| Halina   | Ja, ook prima, ik ben ook goed**bezig**. Lekker weer hè?                                                                                                 |
| Mercedes | Ja, eindelijk zon. En het wordt morgen ook**warm**! Zeg, hoe**is het met**je moeder? Woont ze nog steeds in Den Haag?                               |
| Halina   | Ja, ze woont daar nog steeds. Ze is al tachtig, maar het gaat goed. Mijn zus woont in de buurt. En mijn broer woont ook niet ver, in Scheveningen. Dat is fijn. |
| Mercedes | Ja, natuurlijk. Doe je moeder**de groeten**.                                                                                                              |
| Halina   | Zal ik doen.**Doei**!                                                                                                                                     |
