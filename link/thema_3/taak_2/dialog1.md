**Rashida heeft een baby**

*Miriam gaat op bezoek bij Rashida, een vriendin. Rashida heeft een baby, een [jongen](xref:225628).*

| Rashida | Ha Miriam, leuk je te zien!                                                                                        |
| ------- | :----------------------------------------------------------------------------------------------------------------- |
| Miriam  | Ja, fijn! Je[ziet er](xref:244779)goed [uit](xref:244779).                                                               |
| Rashida | Dank je.[Kom binnen](xref:237499). [Geef](xref:237703)je [jas](xref:225616)maar hier. Hoe is het?                           |
| Miriam  | Prima hoor. Wat een leuk huis heb je toch. Maar ik zie geen baby. Waar is de[kleine](xref:237775)man?                 |
| Rashida | Hij[slaapt](xref:237955)nog. Hij slaapt veel, hij [drinkt](xref:237559)goed. Hij is heel[lief](xref:237847)!                |
| Miriam  | Nou, dat is fijn. De baby van mijn zus[huilt](xref:237751)veel en slaapt niet goed. Mijn zus is heel [moe](xref:237895). |
| Rashida | O, dat is vervelend. Ik ben niet moe.                                                                              |
| Miriam  | Hoe oud is hij nu? Twee maanden?                                                                                   |
| Rashida | Nee joh, drie maanden.                                                                                             |
| Miriam  | Drie maanden alweer? Dat gaat snel!                                                                                |
| Rashida | Ja, joh, hij[lacht](xref:237811)al veel. Lief, hè? Kom, ga [zitten](xref:238111). Wil je iets drinken?                  |
| Miriam  | Ja, graag.                                                                                                         |
| Rashida | Wat wil je?[Koffie](xref:225652), [thee](xref:225748)?                                                                   |
| Miriam  | Het maakt me niet uit. Wat drink je[zelf](xref:238087)?                                                               |
| Rashida | Ik heb wel zin in koffie.                                                                                          |
| Miriam  | Nou, dan neem ik ook koffie.                                                                                       |
| Rashida | [Zwart](xref:238147) of met [melk](xref:225676)?                                                                         |
| Miriam  | Zwart graag,[zonder](xref:238135) [suiker](xref:225736).                                                                 |
| Rashida | En wil je ook iets lekkers?                                                                                        |
| Miriam  | Nee, dank je. Doe maar niet.                                                                                       |
| Rashida | Oké, ik ga even koffie maken.                                                                                     |
| Miriam  | Hé, ik[hoor](xref:237739)iets!                                                                                       |
| Rashida | Ja, ik ook! Kom!                                                                                                   |
