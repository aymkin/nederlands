Wat is de regel?

Het werkwoord, presens, singularis.

| **werken**              | **sporten**   | **wonen**               | **spellen**             |
| ----------------------------- | ------------------- | ----------------------------- | ----------------------------- |
| **ik** werk             | **ik** sport  | **ik** woon             | **ik** spel             |
| **je** werk**t**  | **je** sport  | **je** woon**t**  | **je** spel**t**  |
| **u** werk**t**   | **u** sport   | **u** woon**t**   | **u** spel**t**   |
| **hij** werk**t** | **hij** sport | **hij** woon**t** | **hij** spel**t** |
| **ze** werk**t**  | **ze** sport  | **ze**woon**t**   | **ze** spel**t**  |

Let op:

| spor**t**en | je spor**t** |
| ----------------- | ------------------ |
| w**o**nen   | ik w**oo**n  |
| spe**ll**en | ik spe**l**  |


Wat is de regel?

Het werkwoord, presens, singularis + je.

**Je** werk**t **in Amsterdam. → Waar werk ** je** ?

**Je** woon**t** in Utrecht. → [Woon** je** in Utrecht?]()

**Je **sport~**t**~ in Amersfoort. → Sport** je** in Amersfoort?

Let op: werk**t** → werk


Wat is de regel?

Het werkwoord, presens, pluralis.

| **werken**                  | **sporten**                  | **wonen**                  | **spellen**                  |
| --------------------------------- | ---------------------------------- | -------------------------------- | ---------------------------------- |
| **we** werk**en**     | **we** sport**en**     | **we** won**en**     | **we** spell**en**     |
| **jullie** werk**en** | **jullie** sport**en** | **jullie** won**en** | **jullie** spell**en** |
| **ze** werk**en**     | **ze** sport**en**     | **ze** won**en**     | **ze** spell**en**     |

Let op:

| Anna werkt.           | **Ze** werk **t** .  |
| --------------------- | -------------------------------- |
| Anna en David werken. | **Ze** werk **en** . |

Het werkwoord ‘hebben’, presens.

| **singularis**             | **pluralis**                |
| -------------------------------- | --------------------------------- |
| **ik** **heb**       | **we** **hebben**     |
|                                  |                                   |
| **je** **hebt**      |                                   |
| **heb~[t]() ~** **je**? |                                   |
|                                  | **jullie** **hebben** |
| **u** **hebt/heeft** |                                   |
|                                  |                                   |
| **hij** **heeft**    | **ze** **hebben**     |

Wat is de regel?

Het werkwoord ‘zijn', presens.

| **singularis**       | **pluralis**              |
| -------------------------- | ------------------------------- |
| **ik** **ben** | **we** **zijn**     |
| **je** bent          |                                 |
|                            | **jullie** **zijn** |
| **u** **bent** |                                 |
|                            |                                 |
| **hij** **is** |                                 |
|                            | **ze** **zijn**     |
| **ze** **is**  |                                 |

Let op:

| Anna**is** in Haarlem.            | **Ze** **is** in Haarlem.   |
| --------------------------------------- | --------------------------------------- |
| Anna en David**zijn** in Haarlem. | **Ze** **zijn** in Haarlem. |




Wat is de regel?

De zin met één werkwoord.

| **1**  | **2**     | **3** | **de rest**  |
| ------------ | --------------- | ----------- | ------------------ |
| **Ik** | **kook**  | pasta.      |                    |
| **Je** | **kookt** | pasta.      |                    |
| **We** | **koken** | pasta       | in het weekend.    |
| **Ze** | **koken** | pasta       | in het restaurant. |

Het **werkwoord** staat op de tweede plaats.


Wat is de regel?

De vraagzin met een vraagwoord.

| **1** | **2**     | **3**          | **4** |
| ----------- | --------------- | -------------------- | ----------- |
| Wie         | **ben**   | **je**?        |             |
| Wat         | **is**    | **het adres**? |             |
| Waar        | **komen** | **jullie**     | vandaan?    |

1 = vraagwoord
2 = **werkwoord**
3 = **subject**

4 = rest van de zin
