## Блок 1: Глаголы в настоящем времени (Het werkwoord in de tegenwoordige tijd)

Настоящее время (presens / tegenwoordige tijd) — основная глагольная форма в
нидерландском. В этом блоке — правила спряжения правильных глаголов и двух
неправильных: _hebben_ и _zijn_.

---

### 1.1 Singularis — единственное число (§1.1)

**Основа глагола (stam)** = инфинитив минус `-en`:

| Infinitief | Stam       | Правило                                  |
| ---------- | ---------- | ---------------------------------------- |
| werk**en** | werk       | Просто убираем -en                       |
| sport**en** | sport     | Просто убираем -en                       |
| won**en**  | woon       | Открытый слог → удваиваем гласную        |
| spell**en** | spel      | Закрытый слог → убираем удвоенную согласную |

**Спряжение:**

| Местоимение | Форма          | Окончание |
| ----------- | -------------- | --------- |
| **ik**      | werk           | stam      |
| **je**      | werk**t**      | stam + t  |
| **u**       | werk**t**      | stam + t  |
| **hij**     | werk**t**      | stam + t  |
| **ze** (ед.)| werk**t**      | stam + t  |

> **Правило:** _ik_ = основа без окончания; все остальные лица ед. числа = основа + **-t**.

#### Правила орфографии (spellingregels)

Это ключевой момент, из-за которого делают много ошибок. В нидерландском
действуют строгие правила написания гласных и согласных:

**1. Открытый / закрытый слог и долгие гласные:**

В инфинитиве `wonen` слог `wo-` открытый (заканчивается на гласную), поэтому
одной буквы `o` достаточно для долгого звука. Но в основе `won` слог закрытый
(заканчивается на согласную `n`), и одна `o` читалась бы как краткий звук. Чтобы
сохранить долгий звук, удваиваем гласную: **woon**.

| Infinitief | Stam   | Почему                                          |
| ---------- | ------ | ----------------------------------------------- |
| wonen      | w**oo**n | Долгая /oː/ в закрытом слоге → удваиваем `o`  |
| lezen      | l**ee**s | Долгая /eː/ в закрытом слоге → удваиваем `e`  |
| maken      | m**aa**k | Долгая /aː/ в закрытом слоге → удваиваем `a`  |

**2. Удвоенные согласные:**

Удвоенная согласная в инфинитиве нужна, чтобы предыдущая гласная оставалась
краткой. В основе слог и так закрытый, поэтому вторая согласная не нужна:

| Infinitief  | Stam  | Почему                                 |
| ----------- | ----- | -------------------------------------- |
| spe**ll**en | spe**l** | Краткая /ɛ/ и так в закрытом слоге  |
| zi**tt**en  | zi**t**  | Краткая /ɪ/ и так в закрытом слоге  |
| he**bb**en  | he**b**  | Краткая /ɛ/ и так в закрытом слоге  |

**3. Озвончение: v→f, z→s:**

Если основа заканчивается на `v` или `z`, в написании они заменяются на глухие
`f` и `s` (в конце слога нидерландский не пишет звонкие):

| Infinitief | Stam    |
| ---------- | ------- |
| le**z**en  | lee**s** |
| le**v**en  | lee**f** |
| gelo**v**en| geloo**f** |
| rei**z**en | rei**s** |

#### Примеры в контексте

- _Ik **werk** in Amsterdam._ — Я работаю в Амстердаме.
- _Hij **sport** elke dag._ — Он занимается спортом каждый день.
- _Ze **woont** in Utrecht._ — Она живёт в Утрехте.
- _U **spelt** het goed._ — Вы произносите по буквам правильно.

---

### 1.2 Inversie met 'je' — инверсия с 'je' (§1.2)

Когда подлежащее `je` стоит **после** глагола (в вопросах или при инверсии),
окончание **-t отпадает**:

| Прямой порядок         | Инверсия / вопрос       |
| ---------------------- | ----------------------- |
| **Je** werk**t** in Amsterdam. | Waar werk **je**?    |
| **Je** woon**t** in Utrecht.   | Woon **je** in Utrecht? |
| **Je** sport in Amersfoort.    | Sport **je** in Amersfoort? |

> **Правило:** При инверсии с _je_ глагол теряет `-t`.
>
> Если основа уже заканчивается на `-t` (например, _sport_), ничего не меняется — форма и так без дополнительного `-t`.

#### Почему это важно?

Это правило чисто фонетическое — _werkt je_ звучит неуклюже с двумя
взрывными подряд. Язык «экономит» и убирает лишний звук.

#### Примеры

- _**Je** **spreekt** goed Nederlands._ → _**Spreek je** goed Nederlands?_
- _**Je** **fietst** naar je werk._ → _**Fiets je** naar je werk?_
- _**Je** **leest** veel boeken._ → _**Lees je** veel boeken?_

---

### 1.3 Pluralis — множественное число (§1.3)

Множественное число — самое простое: для всех лиц форма = **инфинитив**:

| Местоимение | Форма        |
| ----------- | ------------ |
| **we**      | werk**en**   |
| **jullie**  | werk**en**   |
| **ze** (мн.)| werk**en**   |

> **Правило:** множественное число presens = инфинитив (стam + en).

#### Отличие ze (ед.) от ze (мн.)

Местоимение `ze` может означать и «она» (ед.), и «они» (мн.). Различить можно
только по форме глагола:

| Предложение                 | ze = ?        |
| --------------------------- | ------------- |
| Anna werkt. **Ze** werk**t**. | она (singularis) |
| Anna en David werken. **Ze** werk**en**. | они (pluralis)   |

#### Примеры

- _We **wonen** in Den Haag._ — Мы живём в Гааге.
- _Jullie **sporten** veel._ — Вы много занимаетесь спортом.
- _Ze **werken** in het restaurant._ — Они работают в ресторане.

---

### 1.4 Глагол 'hebben' — иметь (§1.5)

Неправильный глагол. Нужно запоминать:

| Singularis                | Pluralis                |
| ------------------------- | ----------------------- |
| **ik** heb                | **we** hebben           |
| **je** hebt               | **jullie** hebben       |
| **u** hebt / heeft        |                         |
| **hij** heeft             | **ze** hebben           |
| **ze** (ед.) heeft        |                         |

**Особенности:**

- _ik heb_ — без `-t` (как у правильных глаголов)
- _je hebt_ → при инверсии: **heb je?** (теряет `-t`, как по правилу §1.2)
- _u_ допускает две формы: _hebt_ и _heeft_ (обе правильные)
- 3-е лицо: _heeft_ (не ~~hebt~~)

#### Примеры

- _Ik **heb** twee kinderen._ — У меня двое детей.
- _**Heb** je een pen?_ — У тебя есть ручка?
- _Hij **heeft** een hond._ — У него есть собака.
- _Ze **hebben** een groot huis._ — У них большой дом.

---

### 1.5 Глагол 'zijn' — быть (§1.6)

Самый неправильный глагол — все формы нужно запоминать:

| Singularis            | Pluralis              |
| --------------------- | --------------------- |
| **ik** ben            | **we** zijn           |
| **je** bent           | **jullie** zijn       |
| **u** bent            |                       |
| **hij** is            | **ze** zijn           |
| **ze** (ед.) is       |                       |

**Особенности:**

- Четыре разных основы: _ben_, _bent_, _is_, _zijn_
- _u bent_ (не ~~u is~~, в отличие от разговорного)
- В разговорной речи _je bent_ часто сокращается: _je bent_ → _je **ben**_ (неформально)

#### Примеры

- _Ik **ben** student._ — Я студент.
- _**Ben** je moe?_ — Ты устал?
- _Hij **is** Nederlands._ — Он нидерландец.
- _We **zijn** thuis._ — Мы дома.
- _Ze **is** in Haarlem._ — Она в Харлеме.
- _Ze **zijn** in Haarlem._ — Они в Харлеме.

## Блок 2: Порядок слов (De woordvolgorde)

Порядок слов в нидерландском — строгий. Глагол в главном предложении всегда
стоит на **втором месте** (V2-правило). Это фундаментальная черта языка.

---

### 2.1 Утвердительное предложение с одним глаголом (§2.4)

| Позиция 1      | Позиция 2 (глагол) | Позиция 3+    |
| --------------- | ------------------- | ------------- |
| **Ik**          | **kook**            | pasta.        |
| **Je**          | **kookt**           | pasta.        |
| **We**          | **koken**           | pasta in het weekend. |
| **Ze**          | **koken**           | pasta in het restaurant. |

> **Правило:** глагол **всегда** на 2-й позиции.

Позиция 1 — обычно подлежащее, но может быть и обстоятельство (тогда
подлежащее сдвигается за глагол — это называется **инверсия**):

| Позиция 1          | Позиция 2 (глагол) | Позиция 3 (субъект) | Остальное |
| ------------------- | ------------------- | ------------------- | --------- |
| **'s Ochtends**     | **kook**            | **ik**              | pasta.    |
| **In het weekend**  | **koken**           | **we**              | pasta.    |

#### Примеры

- _Ik **werk** in Amsterdam._ — Я работаю в Амстердаме.
- _In Amsterdam **werk** ik._ — В Амстердаме я работаю. (инверсия)
- _We **koken** pasta in het weekend._ — Мы готовим пасту в выходные.

---

### 2.2 Вопрос с вопросительным словом (§2.6)

| Позиция 1 (vraagwoord) | Позиция 2 (глагол) | Позиция 3 (субъект) | Остальное |
| ----------------------- | ------------------- | ------------------- | --------- |
| **Wie**                 | **ben**             | **je**?             |           |
| **Wat**                 | **is**              | **het adres**?      |           |
| **Waar**                | **komen**           | **jullie**          | vandaan?  |

> **Правило:** вопросительное слово на 1-м месте, глагол на 2-м, подлежащее на 3-м.

**Вопросительные слова (vraagwoorden):**

| Vraagwoord | Перевод       | Пример                            |
| ---------- | ------------- | --------------------------------- |
| wie        | кто           | _Wie **is** dat?_                 |
| wat        | что           | _Wat **doe** je?_                 |
| waar       | где           | _Waar **woon** je?_               |
| wanneer    | когда         | _Wanneer **kom** je?_             |
| waarom     | почему        | _Waarom **leer** je Nederlands?_  |
| hoe        | как           | _Hoe **heet** je?_                |
| welk(e)    | какой/какая   | _Welke taal **spreek** je?_       |

#### Да/нет-вопросы (ja/nee-vragen)

Хотя в §2.6 рассматриваются только wh-вопросы, полезно знать: в вопросах
без вопросительного слова глагол стоит на **1-м месте**:

| Позиция 1 (глагол) | Позиция 2 (субъект) | Остальное     |
| ------------------- | ------------------- | ------------- |
| **Woon**            | **je**              | in Utrecht?   |
| **Spreek**          | **je**              | Nederlands?   |
| **Ben**             | **je**              | student?      |

## Блок 3: Личные местоимения (Personale pronomina)

---

### 3.1 Первое и второе лицо (§6.1)

| Singularis | Перевод              | Pluralis   | Перевод         |
| ---------- | -------------------- | ---------- | --------------- |
| **ik**     | я                    | **we**     | мы              |
| **je**     | ты (неформально)     | **jullie** | вы (неформально)|
| **u**      | вы (формально)       |            |                 |

**Нюансы:**

- **je** — стандартная неформальная форма «ты». В письменном языке иногда
  используется полная форма **jij** (для ударения/контраста):
  _**Jij** werkt hier, niet ik._ — Это **ты** тут работаешь, не я.

- **u** — вежливая форма. Используется с незнакомыми людьми, в формальной
  обстановке, с пожилыми людьми. Глагол при _u_ спрягается как 3-е лицо
  (с `-t`): _U **spreekt** goed Nederlands._

- **we** — разговорная форма. Полная форма — **wij** (для контраста):
  _**Wij** gaan naar Italië, en jullie?_ — **Мы** едем в Италию, а вы?

- **jullie** — множественное «вы» в неформальной обстановке. Формального
  множественного «вы» нет — используется тоже **u**.

---

### 3.2 Третье лицо (§6.2)

| Singularis | Используется для       | Pluralis | Используется для |
| ---------- | ---------------------- | -------- | ---------------- |
| **hij**    | мужчина / de-слово     | **ze**   | любой мн. род    |
| **ze**     | женщина / de-слово     |          |                  |
| **het**    | het-слово (предмет)    |          |                  |

**Примеры из учебника:**

- _Dit is Ali. **Hij** is een man._ — Это Али. **Он** мужчина.
- _Dit is Sarah. **Ze** is een vrouw._ — Это Сара. **Она** женщина.
- _Dit zijn Ali en Sarah. **Ze** zijn getrouwd._ — Это Али и Сара. **Они** женаты.
- _Dit zijn Mohamed, Adam en Sofia. **Ze** eten pasta._ — Это Мохамед, Адам и
  София. **Они** едят пасту.

**Нюанс: ze = она ИЛИ они**

Как мы уже видели в §1.3, различить «она» и «они» можно по форме глагола:

| ze + глагол ед. ч. | = она  | _Ze **woont** in Utrecht._   |
| ------------------- | ------ | ---------------------------- |
| ze + глагол мн. ч.  | = они  | _Ze **wonen** in Utrecht._   |

---

### 3.3 Сводная таблица местоимений (§6.3)

|               | Singularis | Pluralis   |
| ------------- | ---------- | ---------- |
| **1-е лицо**  | ik         | we (wij)   |
| **2-е лицо**  | je (jij)   | jullie     |
| **2-е форм.** | u          | u          |
| **3-е муж.**  | hij        |            |
| **3-е жен.**  | ze (zij)   | ze (zij)   |
| **3-е ср.**   | het        |            |

**Полные и краткие формы:**

В нидерландском у большинства местоимений есть **ударная** (полная) и
**безударная** (краткая) форма:

| Краткая (обычная) | Полная (ударная) | Когда полная?                    |
| ----------------- | ---------------- | -------------------------------- |
| ik                | —                | Не имеет полной формы            |
| je                | **jij**          | Контраст, ударение               |
| we                | **wij**          | Контраст, ударение               |
| ze (ед.)          | **zij**          | Контраст, ударение               |
| ze (мн.)          | **zij**          | Контраст, ударение               |

_**Zij** is de docent, niet hij._ — **Она** преподаватель, не он.

## Шпаргалка: алгоритм спряжения в presens

```
1. Берём инфинитив:          werken, wonen, lezen, zitten
2. Убираем -en → основа:     werk, woon*, lees*, zit*
3. Применяем орфографию:
   - Долгая гласная в закрытом слоге → удваиваем: won → woon
   - Удвоенная согласная → убираем одну:          zitt → zit
   - v → f, z → s на конце:                       lez → lees (+ удвоение)
4. Спрягаем:
   - ik         = основа
   - je/u/hij/ze = основа + t
   - we/jullie/ze (мн.) = инфинитив
5. Проверяем:
   - Основа на -t? → je/u/hij/ze тоже без доп. t (sport → je sport)
   - Инверсия с je? → убираем -t (werkt je → werk je)
```
