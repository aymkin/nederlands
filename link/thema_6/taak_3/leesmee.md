# Leesmee

## Ik heb vertraging

Het is vijf voor negen uur. De Nederlandse les begint om negen uur.

De telefoon van Rami gaat.

Rami Hé, Ali.

Ali Hoi, ik zit nog in de trein. Ik heb vertraging. Ik ben over ongeveer een
half uur op school, denk ik.

Zeg jij het tegen de docent?

Rami Ik lig in bed, man, ik ben ziek. Ik ga niet naar de les.

Ali O, wat heb je?

Rami Gewoon, ik denk griep.

Ali Oh, vervelend voor je. Nou ja, dan stuur ik wel een appje. Ik spreek je
later.

Rami Oké.

## Ik sta in de file

Het is kwart voor negen. Mary zit in haar auto. Ze kan niet op tijd op haar werk
komen. Ze belt naar haar manager, Paul.

Paul Met Paul.

Mary Goedemorgen, Paul.

Paul Ha Mary! Wat is er? Waarom bel je?

Mary Sorry, ik kom wat later. Ik sta in de file op de A1. Een enorme file na een
ongeluk. Heel vervelend.

Paul Ja, dat is zeker vervelend. Straks is toch die vergadering over de
verhuizing van het kantoor?

Mary Dat klopt. Die begint om negen uur, maar ik ben pas over ongeveer een uur
op het werk.

Paul Nou ja, daar is niets aan te doen. Ik zeg het wel tegen de collega’s.
Sterkte!

Mary Dank je! Ik ga weer rijden!

## Werk samen. Lees de zinnen hardop.

A begint. Wissel daarna van rol.

A: Hé, Ali B: Hoi, ik zit nog in de trein. Ik heb een half uur vertraging. Zeg
jij het tegen de docent? A: Ik lig in bed, man, ik ben ziek. Ik ga niet naar de
les. B: O, wat heb je? A: Griep, denk ik. B: Oh, vervelend voor je. Nou ja, dan
stuur ik wel een appje. Ik spreek je later. A: Oké.

A: Met Paul. B: Goedemorgen, Paul. A: Ha Mary! Wat is er? Waarom bel je? B:
Sorry, ik kom wat later. Ik sta in de file op de A1. Heel vervelend. A: Ja, dat
is zeker vervelend. Straks is toch die vergadering? B: Dat klopt. Die begint om
negen uur, maar ik ben pas om tien uur op het werk. A: Nou ja, niets aan te
doen. Ik zeg het wel tegen de collega’s. Sterkte! B: Dank je! Ik ga weer rijden!

## Werk samen. Bespreek samen.

Wat kun je schrijven? Kies a of b.

1. John zit in de trein. Hij heeft een afspraak bij de gemeente. Hij kan niet op
   tijd komen. a. Beste meneer Keizer, helaas kom ik wat later. Mijn trein heeft
   vertraging. Excuses! John Davids. b. Hoi! Ik kom wat later. Sorry! Groet,
   John.

2. Peter zit in de bus. Zijn vrouw is thuis. De bus staat in een file. a. Beste
   Sue, ik kom wat later thuis. Mijn excuses. Peter van Kampen. b. Hoi schat, ik
   sta in de file. Ik kom wat later. Tot zo! Kusjes, Peter.
