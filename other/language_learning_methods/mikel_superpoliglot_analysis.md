# Анализ метода Mikel Superpoliglot

> Стратегическая оценка, сравнение с другими методами, применимость для
> нидерландского

---

## Структура метода

| Шаг       | Название                   | Суть                                                               | Время                 |
| --------- | -------------------------- | ------------------------------------------------------------------ | --------------------- |
| **1**     | Language Islands           | Записать все свои реальные фразы → перевести AI                    | Подготовка (3-5 дней) |
| **2**     | Audio flooding + Shadowing | TTS-аудио своих фраз → слушать в «мёртвое время» + повторять вслух | Весь день (фоново)    |
| **3**     | Active Recall              | Родная фраза → произнести перевод вслух → проверить                | 30-60 мин/день        |
| **Бонус** | Pre-input Comprehension    | Взять транскрипт видео → выучить слова → смотреть                  | По желанию            |

---

## Стратегические принципы

### 1. Персонализация контента (Language Islands)

Ключевое отличие от всех других методов. Вместо учебника, Duolingo или даже
мотивированного выбора из видео (Борисов) — ты записываешь **буквально всё, что
говоришь за день**, и это становится учебным материалом. Логика: зачем учить
слово «фламинго», если ты каждый день говоришь «созвон перенесли на три»?

### 2. Фразы, не слова

Совпадает с Борисовым. Оба против изолированных слов. Но у Микеля фокус на
_своих_ фразах, у Борисова — на фразах _из контента_.

### 3. AI как инструмент

Микель открыто использует AI для перевода и TTS. Это технически ближе всего к
тому, что уже есть в репозитории (`text_to_speech.py`, `audio_to_anki.py`).

### 4. Мёртвое время = учебное время

Как и Алишер: дорога, душ, тренажёрка. Но аудио не случайное — строго свои
фразы.

### 5. Shadowing как мост input → output

Совпадает с Борисовым (shadowing за носителем) и методом Евгения (шаг 4). Но у
Микеля shadowing делается по _своим_ фразам, а не по чужому контенту.

### 6. Active recall = ядро системы

Полностью совпадает с Борисовым («активное припоминание бетонирует нейронную
связь»). Разница: Борисов делает это через карточки/Anki, Микель — устно, без
карточек как таковых.

## Что уникального у Микеля

1. **Language Islands** — концепция «личных островов» не встречается ни у кого
   другого. Самый персонализированный подход: ты учишь буквально _свой_ язык.

2. **Pre-input Comprehension** — учить слова ДО просмотра видео, а не после.
   Борисов и Алишер работают наоборот: смотришь → выписываешь непонятное. Микель
   переворачивает: выучи → смотри → закрепи.

3. **Агрессивный таймлайн** — 6 недель до conversational. Борисов закладывает
   год (1000 часов), Алишер — 3-6 месяцев только на input. Микель обещает
   результат быстрее, но за счёт узкого фокуса на _своих_ темах.

## Применимость для Alex (A2, нидерландский, Hilversum)

| Техника Микеля              | Применимость | Как реализовать                                |
| --------------------------- | ------------ | ---------------------------------------------- |
| **Language Islands**        | Высокая      | Speech-to-text → перевод → `*_anki.txt`        |
| **TTS аудио**               | Уже есть!    | `text_to_speech.py` для генерации аудио        |
| **Shadowing своих фраз**    | Высокая      | В дороге Hilversum → Amsterdam                 |
| **Active recall устно**     | Высокая      | Без карточек, просто вслух по списку           |
| **Pre-input comprehension** | Средняя      | Скачать транскрипт → выучить → смотреть        |
| **Self-narration**          | Высокая      | Описывать день на NL вслух (дома, на прогулке) |

## Стратегическая оценка

### Сильные стороны

- Максимальная практичность: учишь то, что реально скажешь
- Низкий порог входа: 30 мин фокуса + фоновое время
- Отлично совместим с инструментами репозитория (TTS, Anki)
- Быстрый первый результат (мотивация не падает)

### Слабые стороны

- «Пузырь» своих тем: без широкого input не поймёшь, что говорят _тебе_
- Нет comprehensible input фазы — Борисов и Алишер правы, что без неё модель
  языка будет фрагментарной
- Агрессивный timeline (6 недель) реалистичен только для близких языков (ES, PT
  для IT-говорящих), для NL/RU — скорее 3-4 месяца
- Self-narration без коррекции может закрепить ошибки (об этом предупреждает
  Алишер)

### Вердикт

Микель даёт мощную _тактику_ (Language Islands, pre-input comprehension), но как
самостоятельная _стратегия_ он неполон — нужен широкий input (Борисов/Алишер)
для понимания. Лучше всего работает как **дополнение** к текущему методу Евгения
6 шагов: Language Islands для шагов 1-3, а pre-input comprehension — для
вечернего просмотра новых видео.

### Ключевые выводы для практики

1. Language Islands — это по сути то, что метод Евгения уже делает на шаге 3
   (извлечение фраз), но Микель расширяет источник: не только из видео, но из
   _всей своей жизни_
2. Pre-input comprehension отлично ложится на вечерний цикл: перед «холодным
   просмотром» (шаг 1 Евгения) изучить транскрипт — и шаг 1 перестаёт быть
   «холодным», а становится закреплением
3. Из всех методов в коллекции именно Микель лучше всего «дружит» с
   AI-инструментами (TTS, перевод, speech-to-text)
