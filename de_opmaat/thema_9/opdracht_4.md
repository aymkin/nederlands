# 4 Spreken

Werk in tweetallen. Voer de volgende telefoongesprekken.

1 (A) Je hebt over een uur een afspraak met B. Je hebt de trein gemist en komt
later. (B) Je hebt over een uur een afspraak met A. Vraag of je A kan ophalen
van het station.

Oplossing: (A) Hallo, met Oleksandr Naumkin. Mag ik met Bob spreken? (B) Met
Bob. Hoe kan ik je helpen? (A) Ik heb een afspraak met je over een uur. Helaas,
kan ik niet op tijd komen, want ik heb de trein gemist. (B) Kun je iets later
komen? (A) Jazeker. Over een uur, denk ik. (B) Kan ik je van het station
ophalen? (A) Prima hoor! (B) Goed, tot straks dan! (A) Tot zo!

2 (A) Je belt het taalinstituut. Je bent ziek en komt vandaag niet. Je docent
heet Tom. (B) Je bent receptionist bij een taalinstituut. Je neemt de boodschap
aan.

Oplossing: (B) Met taalinstituut Europa. Hoe kan ik je helpen? (A) Mijn naam is
Alex. Ik ben namelijk ziek vandaag. Dus kan ik niet komen. (B) Ja, hoor. Wat is
je groep? Wie is je docent? (A) Mijn groep is "Niveau Drie" en mijn docent is
Tom. (B) Wacht maar even een beetje... Ja, ik zie het. Ik ga de boodschap aan
docent Tom doorgeven. (A) Hartelijk bedankt. Dag! (B) Beterschap, tot ziens!

3 (A) Je belt bakkerij Iris. Je hebt een bestelling voor zaterdag. Bedenk zelf
wat je wilt bestellen. (B) Je werkt bij bakkerij Iris. Neem de bestelling aan en
spreek een tijd af.

Oplossing: (A) Hallo, mag ik met Iris spreken? (B) Ja, natuurlijk, met Iris. Hoe
kan ik je helpen? (A) Ik wil graag een paar broodjes bestellen. Hoe kan ik het
doen? (B) Zeg het maar. (A) Jazeker. Zo, ik wil graag twee berlinerbollen met
chocolade, vier broodjes met kaas en een croissant. (B) Ja, dat is prima, nog
anders? (A) Nee, dank je. (B) Wil je je bestelling ophalen of thuisbezorgen? (A)
Ik kom over drie dagen, op zaterdag om zes uur. (B) Ja, dat werkt goed. (A) Tot
later, dan! (B) Tot ziens!

4 (A) Je hebt kiespijn en belt de tandarts. Je wilt zo snel mogelijk een
afspraak. (B) Je bent tandartsassistente. Stel vragen over de kiespijn: welke
tand/kies? hoelang al kiespijn? Maak een afspraak

Oplossing: (A) Hallo, met Alex Stall, mag ik met mevrouw Boer spreken? (B)
Helaas, ze is nu niet beschikbaar. Ik kan een boodschap aan de arts doorgeven.
(A) Ik heb vreselijke kiespijn. Ik heb hulp nodig. (B) Hoe lang heeft u al pijn?
(A) Sinds gisterenavond. (B) Welke kies doet pijn? (A) De bovenste tand rechts.
(B) Kunt u om half zes komen? (A) Ja, zeker, zo snel mogelijk. (B) Komt u dan
naar ons, wacht niet meer. Tot snel! (A) Dank u, tot ziens.

## Foutenlog — 19.03.2026

### Dialoog 1: afspraak / trein gemist

| Fout                       | Correctie                    | Regel                                      |
| -------------------------- | ---------------------------- | ------------------------------------------ |
| ~~spreeken~~               | spreken                      | Één e in de stam: spre·ken                 |
| ~~How~~ kan ik             | **Hoe** kan ik               | Engels → Nederlands                        |
| ~~I~~ heb                  | **ik** heb                   | Engels → Nederlands                        |
| ~~Good~~                   | **Goed**                     | Engels → Nederlands                        |
| ~~Helas~~                  | **Helaas**                   | Dubbele a: he·laas                         |
| ~~sation~~                 | **station**                  | Vergeet de t niet: sta·ti·on               |
| ~~Kun~~ ik                 | **Kan** ik                   | ik kan / kan ik; kun is voor jij           |
| Kun je ~~kom~~ iets later? | Kun je iets later **komen**? | Infinitief na modaal werkwoord → einde     |
| ~~Kun ik ophalen je~~      | Kan ik **je** ophalen?       | Pronomen vóór infinitief                   |
| met ~~u~~ / met je         | Kies één register            | Niet mengen: u (formeel) of je (informeel) |
