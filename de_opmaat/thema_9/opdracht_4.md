# 4 Spreken

Werk in tweetallen. Voer de volgende telefoongesprekken.

1 
(A) Je hebt over een uur een afspraak met B. Je hebt de trein gemist en komt later.
(B) Je hebt over een uur een afspraak met A. Vraag of je A kan ophalen van het station.

(A) Hallo, met Oleksandr Naumkin. Mag ik met Bob spreken?
(B) Met Bob. Hoe kan ik je helpen?
(A) Ik heb een afspraak met je over een uur. Helaas, kan ik niet op tijd komen, want ik heb de trein gemist.
(B) Kun je iets later komen?
(A) Jazeker. Over een uur, denk ik.
(B) Kan ik je van het station ophalen?
(A) Prima hoor!
(B) Goed, tot straks dan!
(A) Tot zo!

2
(A) Je belt het taalinstituut. Je bent ziek en komt vandaag niet. Je docent heet Tom.
(B) Je bent receptionist bij een taalinstituut. Je neemt de boodschap aan.
(B) Met taalinstituut Europa. Hoe kan ik je helpen?
(A) Mijn naam is Alex. Ik ben namelijk ziek vandaag. Dus kan ik niet komen.
(B) Ja, hoor. Wat is je groep? Wie is je docent?
(A) Mijn groep is "Niveau Drie" en mijn docent is Tom.
(B) Wacht maar even een beetje... Ja, ik zie het. Ik ga de boodschap aan docent Tom doorgeven.
(A) Hartelijk bedankt. Dag!
(B) Beterschap, tot ziens!



3

(B) Je werkt bij bakkerij Iris. Neem de bestelling aan en spreek een tijd af.

4

## Foutenlog — 19.03.2026

### Dialoog 1: afspraak / trein gemist

| Fout | Correctie | Regel |
|------|-----------|-------|
| ~~spreeken~~ | spreken | Één e in de stam: spre·ken |
| ~~How~~ kan ik | **Hoe** kan ik | Engels → Nederlands |
| ~~I~~ heb | **ik** heb | Engels → Nederlands |
| ~~Good~~ | **Goed** | Engels → Nederlands |
| ~~Helas~~ | **Helaas** | Dubbele a: he·laas |
| ~~sation~~ | **station** | Vergeet de t niet: sta·ti·on |
| ~~Kun~~ ik | **Kan** ik | ik kan / kan ik; kun is voor jij |
| Kun je ~~kom~~ iets later? | Kun je iets later **komen**? | Infinitief na modaal werkwoord → einde |
| ~~Kun ik ophalen je~~ | Kan ik **je** ophalen? | Pronomen vóór infinitief |
| met ~~u~~ / met je | Kies één register | Niet mengen: u (formeel) of je (informeel) |

### Dialoog 2: ziekmelding bij taalinstituut

| Fout | Correctie | Regel |
|------|-----------|-------|
| ~~naamelijk~~ | **namelijk** | Één a: na·me·lijk |
| ~~vandag~~ | **vandaag** | Dubbele a: van·daag |
| ~~Niweau~~ | **Niveau** | Frans woord met v: ni·veau |
| ~~eenbetje~~ | **een beetje** | Twee woorden, dubbele e |
| ~~Beeterschap~~ | **Beterschap** | Één e: be·ter·schap |
| ~~Doe!~~ | **Dag!** / **Doei!** | "Doe" = делай; "Doei" = пока |
