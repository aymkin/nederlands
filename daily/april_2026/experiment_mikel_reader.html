Experiment: Метод Микеля

Language Islands + Pre-input Comprehension
Период: 14–30 апреля 2026
Mikel Superpoliglot — адаптация

Гипотеза

Ежедневная запись повседневных фраз (self-talk) с переводом на нидерландский даст более быстрое усвоение личной лексики, чем изучение по учебнику.

Не гонимся за 6 неделями. Цель — построить базу из личных тем («о себе»), которая станет фундаментом для грамматики, лексики, паттернов. Фрагментарно в начале — потом строим мосты между островами.

Два потока

Поток 1: Language Islands (ежедневно)

  1. В течение дня записывать self-talk через speech-to-text (русский)
  2. Вечером/следующим утром — принести текст в Claude
  3. Claude: очищает, переводит, создаёт файл + Anki карточки
  4. Учить: shadowing + active recall по своим фразам

Поток 2: Pre-input Comprehension (1–2 раза в неделю)

  1. Выбрать видео на нидерландском
  2. Скачать субтитры → нормализовать → reader.html
  3. Прочитать текст на телефоне, пометить незнакомое
  4. Выучить незнакомые слова
  5. Посмотреть видео — теперь ~70–80% понятно

Daily workflow

Утро:      Shadowing вчерашних фраз (в дороге, 15-20 мин)
День:      Записывать self-talk (speech-to-text, фоново)
Вечер:     Принести текст → Claude → новые islands
           Active recall по всем islands (10-15 мин)
Перед сном: Anki — islands карточки (10 мин)
