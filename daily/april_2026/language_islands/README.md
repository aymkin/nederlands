# Language Islands — Daily Workflow

> Метод Микеля: записывай свою жизнь → переводи → учи

## Как записывать

1. В течение дня используй **speech-to-text** на телефоне (любое приложение)
2. Говори с собой на русском обо всём: работа, еда, мысли, планы, эмоции
3. Не фильтруй — записывай всё подряд, даже бытовое
4. Вечером скопируй текст

## Как обрабатывать

1. Принеси текст в Claude
2. Скажи: «Вот мой self-talk за сегодня, обработай как Language Islands»
3. Claude:
   - Очистит от мусора speech-to-text
   - Разобьёт на фразы
   - Переведёт на нидерландский (живой, не учебниковый)
   - Добавит грамматические пометки
   - Допишет в Anki файл

## Как учить

- **Утро (дорога):** shadowing — читай вслух вчерашние фразы
- **День:** active recall — закрой перевод, попробуй сказать на NL
- **Вечер:** Anki — повтори карточки `islands_april_anki.txt`

## Формат файлов

- `2026-04-DD.md` — дневной файл с таблицами RU → NL
- `islands_april_anki.txt` — накопительный Anki (TSV, 4 колонки)

## Anki: Note Type «Language Islands»

### Первая настройка (один раз)

1. **Tools → Manage Note Types → Add**
2. Выбери «Add: Basic (and reversed card)» → назови **Language Islands**
3. **Fields:** переименуй в `Nederlands`, `Русский`, `Grammatica`
4. **Cards:** вставь шаблоны из этого README (ниже)

### Импорт карточек

1. **File → Import** → выбери `islands_april_anki.txt`
2. Note Type: **Language Islands**
3. Поля маппятся автоматически (через `#columns` в файле)
4. При повторном импорте — Anki обновит существующие, добавит новые

### Две карточки на каждую фразу

| Карточка | Направление | Цель |
| --- | --- | --- |
| Card 1 | 🇷🇺 → 🇳🇱 | Active recall: видишь русский, говоришь на NL |
| Card 2 | 🇳🇱 → 🇷🇺 | Recognition: видишь NL, понимаешь смысл |

### Теги

Формат: `islands::april::dag_NN::topic`

Темы: `werk`, `eten`, `sport`, `dagelijks`, `sociaal`

Фильтрация в Anki: `tag:islands::april::dag_13` — все фразы за день
